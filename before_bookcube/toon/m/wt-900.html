SNS 친구 공유 후 적립금 지급 받는 방법

  • 본인인증 완료한 회원만 적립금 지급이 가능합니다.
  • SNS 친구 공유 통한 적립금 (300원) 지급은 1달에 1번만 가능 합니다. (유효기간 30일)
  • 매월초 또는 적립금을 지급 받으시면 해당 월에 SNS 공유한 횟수는 자동 리셋 됩니다.
  • 코믹큐브 로그인 후 카카오 스토리/페이스북/트위터를 이용하여 해당 도서의 링크 주소를 공유 합니다.
  • SNS를 통해 공유 된 링크 주소를 친구 또는 지인분들께서 터치 또는 클릭 할 경우 SNS 공유 횟수로 카운팅 됩니다.
    (비로그인 상태에서 SNS 공유 하면, 공유 횟수가 카운팅 되지 않습니다.)
  • 공유 횟수가 10번을 충족한 경우 적립금 300원이 자동 지급 되며, 알리미 통해서 확인 가능 합니다
