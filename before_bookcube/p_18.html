쿠폰 번호 등록하기
도서 쿠폰 등록 방법 안내
  1. 도서 쿠폰 번호를 등록창에 입력해 주세요.
  2. 입력 후 등록 버튼을 클릭하면 도서 혹은 오디오 파일이 자동으로 지급됩니다.
  3. 쿠폰은 계정당 1번만 등록 할 수 있습니다.(중복등록 불가)
  4. 마이페이지 > 전자책 다운로드 > 10년 소장탭에서 다운로드가 가능합니다.
선물 쿠폰 등록 방법 안내
  1. 선물 받은 이메일/카카오톡에 포함된 쿠폰번호를 등록창에 입력해 주세요.
  2. 입력 후 등록 버튼을 클릭하면 선물 받은 상품(북캐시/정액권/도서)이 자동으로 지급됩니다.
  3. 선물 받은 쿠폰은 하나의 계정에서만 등록 할 수 있습니다.
  4. 받은 선물은 [마이페이지 > 선물함 > 받은선물함]에서 목록 확인이 가능합니다.
  5. 도서를 선물 받은 경우, [마이페이지 > 북큐브 서점 > 내서재] 에서 다운로드가 가능합니다.
적립금 쿠폰 등록 방법 안내
  1. 적립금 쿠폰 번호를 등록창에 입력해 주세요.
  2. 입력 후 등록 버튼을 클릭하면 적립금이 자동으로 지급됩니다.
  3. 쿠폰은 계정당 1번만 등록 할 수 있습니다.(중복등록 불가)
  4. 적립금은 유효기간이 있으며 적립금 지급 및 유효기간은
    마이페이지 > 북캐시/적립금/상품권탭 > 적립금 적립내역에서 확인 할 수 있습니다.
