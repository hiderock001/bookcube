오디오 플레이어

STEP 1 오디오 플레이어 실행

  • 1. 메인 화면에서 오디오 플레이어를 선택한 후, 방향키를 누르거나 메인 화면이 디스플레이 된 상태에서 오디오
    플레이어에 해당하는 2번 숫자 키를 선택하면 바로 실행됩니다.
  • 2. 또는 메인 화면이 디스플레이된 상태에서 북플레이어에 해당하는 1번 숫자 키를 선택하면 바로 실행됩니다.

STEP 2 오디오 플레이어 사용법

  • 1. 한 페이지에 10개의 오디오 파일이 표시되며 듣고자 하는 오디오 파일을 선택하여 [ENTER] 키나 방향 키를 누르면
    해당 파일이 재생되기 시작합니다.
  • 2. 오디오 파일을 재생시킨 후에 [MENU] 키를 눌러서 [8. 전자책 이어보기]를 실행하면 음악을 들으면서 전자책을
    읽을 수 있습니다.
  • 3. 단말기 상단 왼쪽 측면에 위치한 볼륨 조절 키로 음량 조절을 할 수 있으며, 음량 크기는 화면 우측 상단에 아이콘과
    현재의 볼륨 레벨을 나타내는 숫자가 표시됩니다. 볼륨 레벨은 0부터 15(MAX)까지 입니다.
  • 4. 음악 파일 재생의 세부 설정을 하려면 [MENU] 키를 눌러서 원하는 재생 방법을 설정합니다.
  • - 재생 중인 음악 파일이 선택되어 있는 상태에서 다시 [ENTER] 키나 방향 키를 누르면 재생이 중지됩니다.
  • - 오디오 파일은 단말기의 내부 메모리에 저장된 파일명으로 화면에 표시됩니다.
  • - 음악 파일은 MP3 파일과 WMA 파일을 지원합니다.

STEP 3 오디오 플레이어 > 세부 메뉴의 기능

  • 음악 파일이 실행 중일 경우 실행을 중지 시킵니다.
  • 선택한 음악 파일 하나만 재생하고 종료됩니다.
  • 선택한 음악 파일 하나만 반복하여 재생합니다.
  • 단말기 내부 메모리에 저장되어 있는 전체 음악 파일을
    순차적으로 한 번씩 재생하고 모두 재생하면 종료됩니다.
  • 단말기 내부 메모리에 저장되어 있는 전체 음악 파일을 순차적으로
    재생하며, 사용자가 정지시키기 전까지 무한 반복됩니다.
  • 단말기 내부 메모리에 저장되어 있는 전체 음악 파일을 무작위로
    재생하며, 사용자가 정지시키기 전까지 재생합니다.
  • 선택한 음악 파일을 삭제합니다.(재생 중인 음악 파일은 삭제 불가)
  • 음악이 재생된 상태에서 최근까지 보던 전자책을 이어 봅니다.

이미지 플레이어

STEP 1 이미지 플레이어 실행

  • 1. 메인 화면에서 이미지 플레이어를 선택한 후, 방향 키를 누르거나 메인 화면이 디스플레이 된 상태에서 이미지
    플레이어에 해당하는 3번 숫자 키를 선택하면 바로 실행됩니다.
  • 2. 이미지 플레이어를 실행하면 아래 그림과 같이 현재 단말기 내부 메모리 또는 외장 마이크로 SD 카드에 저장되어
    있는 이미지 파일 목록이 표시됩니다.

STEP 2 이미지 플레이어 사용법

  • 1. 읽고자 하는 이미지 파일을 아래 폴더에 저장합니다.
    • - 내부 메모리 : \eBook Store\image
    • - 외장 마이크로 SD카드 : 전체
    • * 모든 이미지들을 폴더 구조로 탐색 가능합니다(JPG, GIF, PNG, BMP, ZIP)
    • * ZIP 파일은 아래와 같은 구조를 지원합니다.
    • - 여러 개의 이미지 파일을 ZIP으로 묶은 경우
    • - 여러 개의 이미지를 포함한 폴더 또는 폴더들을 ZIP으로 묶은 경우
  • 2. 이미지 플레이어를 실행하면 위의 폴더에 있는 이미지들의 목록을 출력합니다.
  • 3. 이미지 파일을 선택하여 실행하면 이미지 플레이어가 실행됩니다.
    • - 이미지가 선택된 상태에서 방향키를 누르거나 "메뉴\실행"을 선택하면 됩니다.
    • - 이미지 출력은 같은 폴더나 ZIP파일 내에서 순서대로 출력되며, 출력 순서는 파일명 또는 파일날짜 순으로
      정렬이 가능합니다.
    • - 이어보기 기능: 이미지 플레이어 종료 후 폴더나 ZIP파일을 재실행하면 자동으로 이전에 읽던 파일이 선택되거나 출력됩니다.
  • 4. 이미지 플레이어 메뉴설명
    • 1. 전체화면 보기(끄기) : 상단의 파일이름과 하단의 페이지 정보가 사라지고 전체화면으로 이미지를 출력
    • 2. 화면에 맞추기: 원본 이미지의 비율을 무시하고 화면 안에 꽉 차게 이미지 출력(전체화면 모드에서도 동일)
    • 3. 자동비율 : 원본 비율대로 화면 안에 이미지를 출력(전체화면 모드에서도 동일)
    • 4. 페이지 이동: 원하는 페이지로 이동

사전

STEP 1 사전 실행

메인 화면에서 사전을 실행(숫자 키 4번)하면 영한사전,
한영사전, 국어사전 중 한 가지를 선택할 수 있는 팝업
메뉴가 나타납니다.

STEP 2 영한사전

  • 1. Prime 영한사전을 선택하여 실행합니다.
  • 2. 영한 사전은 입력한 키워드에 대해 단어, 예문,
    숙어의 세 가지를 검색할 수 있습니다.
  • 3. 단어, 예문, 숙어 검색의 선택을 변경하려면 [Fn]
    키를 누른 상태에서 좌/우 방향 키를 이용합니다.

STEP 3 영한 사전 > 단어 찾기

  • 1. 검색하고자 하는 단어를 키보드의 문자 키를 이용하여 입력합니다.
  • 2. 잘못 입력된 문자는 [DEL] 키를 이용하여 삭제합니다.
  • 3. 입력이 끝나면 [ENTER] 키나 방향 키를 누릅니다.

  • 4. 입력한 키워드로 시작되는 모든 단어들의 목록이 화면에 표시됩니다.
  • 5. 검색하는 단어가 존재할 경우에는 해당 단어가 목록 상단에 표시되며,
    검색 단어가 존재하지 않는 경우에는 가장 비슷한 단어 목록이 알파벳
    순으로 정렬되어 표시됩니다.

  • 6. 단어를 선택하여 [ENTER] 키나 방향키를 누르면 선택한 단어에
    대한 상세 설명이 표시 됩니다.
  • 7. 단어 상세 설명 화면에서 숫자 키 1을 누르면 미국식 발음을 들려주고
    숫자 키 2를 누르면 영국식 발음을 들려줍니다. (단어에 따라서는
    발음이 지원되지 않는 경우도 있습니다.)

STEP 4 영한 사전 > 세부 메뉴의 기능

  • 1. [MENU] 키를 누르면 다른 사전을 실행하거나, 글자크기와
    발음 크기를 설정할 수 있는 팝업 메뉴가 나타납니다. 방향키를
    이용하거나 숫자 키 7번 (글자크기) 혹은 8번(발음크기)를 이용하여
    해당 메뉴를 선택 및 실행합니다.

  • 2. 좌/우 방향 키를 이용하여 원하는 글자 크기를 선택한 후
    [ENTER] 키나 방향 키를 누릅니다.

  • 3. 좌/우 방향 키를 이용하여 원하는 발음크기를 선택한 후
    [ENTER] 키나 방향키를 누릅니다.

STEP 5 한영사전

  • 1. Prime 한영사전을 선택하여 실행합니다.

  • 2. 검색하고자 하는 단어를 키보드의 문자 키들을 이용하여 입력하고
    [ENTER] 키나 방향 키를 누릅니다.

  • 3. 검색한 단어의 철자로 시작되는 단어들의 목록이 출력됩니다.
  • 4. 검색하는 단어가 존재할 경우에는 해당 단어가 화면 상단에 표시되며
    검색 단어가 존재하지 않는 경우에는 가장 비슷한 단어 목록이 철자
    순으로 정렬되어 표시됩니다.

  • 5. 단어를 선택하여 [ENTER] 키나 방향 키를 누르면 단어에 대한
    상세 설명이 표시 됩니다.

STEP 6 국어사전

  • 1. Prime 국어사전을 선택하여 실행합니다.

  • 2. 검색하고자 하는 단어를 키보드의 문자 키들을 이용하여 입력하고
    [ENTER] 키나 방향 키를 누릅니다.

  • 3. 검색한 단어의 철자로 시작되는 단어들의 목록이 출력됩니다.
  • 4. 검색하는 단어가 존재할 경우에는 해당 단어가 화면 상단에 표시되며
    검색 단어가 존재하지 않는 경우에는 가장 비슷한 단어 목록이 철자
    순으로 정렬되어 표시됩니다.

  • 5. 단어를 선택하여 [ENTER] 키나 방향 키를 누르면 단어에 대한
    상세 설명이 표시 됩니다.

환경설정

메인 화면에서 환경설정을 선택하면 우측 하단에 팝업 메뉴가 나타납니다.
PC와 USB로 연결된 유선 연결 상태에서 업데이트를 진행하는 것이 안정적입니다.

업데이트 실행
  • 1. 북플레이어 소프트웨어의 업데이트를 실시간으로 실행합니다.
  • 2. 단말기가 온라인 상태일 때에 업데이트를 할 수 있습니다.
  • 3. 주기적으로 업데이트를 실행하여 최신 프로그램을 사용하는 것이
    좋습니다.
  • 4. 현재 설치된 북플레이어 프로그램에 대한 버전 정보는
    도움말 > 프로그램 정보에서 확인할 수 있습니다.

단말기 고유번호
  • 1. 단말기 고유번호를 조회합니다.
  • 2. 전자책 서점, 전자책 도서관을 이용하기 위해서는 반드시 단말기
    고유번호를 해당 웹 사이트에 등록한 후에 이용하실 수 있습니다.

Wi-Fi 설정 > 전원 ON/OFF
  • 1. 켜기, 끄기를 선택하여 [ENTER] 키가 방향 키를 누르면
    Wi-Fi 전원을 켜거나 끌 수 있습니다. (키패드의 1번과 2번
    키를 선택해도 됩니다.)
  • 2. Wi-Fi 전원이 한 번 켜지면, 단말기의 전원을 끌 때까지는
    ON 상태를 계속 유지하므로, 온라인 접속을 사용하지 않는 경우,
    배터리 관리를 위해 Wi-Fi 전원을 OFF 시켜주시는 것이 좋습니다.
  • 3. Wi-Fi의 전원은 다음 페이지에 설명된 Wi-Fi 접속을 실행할 때
    자동으로 켜지므로 Wi-Fi 전원 ON/OFF 기능은 Wi-Fi 전원을
    OFF 시킬 때 주로 사용합니다.

Wi-Fi 설정 > 자동접속/수동접속
  • 1. 자동 접속으로 설정해두면 비밀번호를 요구하는 AP에 접속할 때,
    이전에 입력해둔 비밀번호를 이용하여 자동으로 접속을 진행하므로
    AP에 처음 접속할 경우에만 비밀번호를 입력하면 됩니다.
  • 2. 수동 접속으로 설정해 두면 비밀번호가 설정된 AP에 접속을 시도할
    때마다 비밀번호를 입력해야 합니다.
  • 3. 설정이 완료되면 닫기로 이동하여 [ENTER] 키나 방향 키를 누르면
    설정 값을 저장한 후 메인 화면으로 이동합니다. (키패드의 5번 키를
    선택해도 됩니다.)

환경설정 > Wi-Fi 접속
  • 1. Wi-Fi 접속하기를 실행하면 Wi-Fi 모듈의 전원이 자동적으로
    켜집니다.
  • 2. 화면 왼쪽 아래에 Wi-Fi 모듈이 ON 상태임을 보여주는
    아이콘()이 나타나면서, 접속 가능한 네트워크의 목록을
    보여 줍니다.
  • 3. 접속하고자 하는 네트워크를 선택한 후[ENTER]나 방향 키를
    누르면 접속을 시도합니다.
  • 4. 보안이라고 표시된 네트워크에 접속하는 경우에는 네트워크
    관리자가 설정한 암호를 입력해야 합니다.
  • 5. 네트워크 연결이 성공하면, 연결된 네트워크 목록에 '연결'이라고
    표시됩니다. Wi-Fi 아이콘에는 수신 감도가 표시됩니다.
